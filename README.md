# Student Management System

A console-based Student Management System developed in Java using Object-Oriented Programming concepts.

## Features

- Add Student
- View All Students
- Search Student by ID
- Delete Student
- Menu Driven Interface

## Concepts Used

- Classes and Objects
- Constructors
- Encapsulation
- Arrays of Objects
- Linear Search
- Array Shifting
- Menu Driven Programming

## Technologies

- Java
- IntelliJ IDEA
